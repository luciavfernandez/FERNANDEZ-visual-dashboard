# Spotify Data Insights Dashboard

## Overview

This dashboard visualizes personal Spotify listening data to uncover behavioral patterns and provide actionable insights for music industry professionals, particularly for timing music releases and optimizing user engagement.

## Project Context

**Target Audience:** Music industry managers and decision-makers  
**Time Commitment:** 10-minute review  
**Data Source:** Personal Spotify listening history

## Dashboard Structure

The dashboard is organized into multiple views:

1. **Overall** - High-level summary of listening behavior
2. **Context** - Background and methodology
3. **Periods** - Temporal analysis (monthly, weekly patterns)
4. **Artists** - Artist-based listening breakdown
5. **Documentation** - Design choices and data explanations
6. **Credits** - Data sources and references

## Why This Matters

Personal data dashboards reveal the **"bigger picture"** of user behavior, especially when real-time data isn't always available to industry professionals. This type of analysis helps:

- **Artists** time their releases strategically
- **Platforms** improve recommendation algorithms
- **Marketers** target campaigns more effectively
- **Product teams** optimize shuffle and autoplay features
